# 📊 Task 3: Data Visualization Dashboard

## Objective

Create an interactive dashboard using Python to visualize key business metrics and trends from the Superstore dataset.

## Dataset

* Superstore Dataset

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dashboard Includes

* KPI Summary
* Sales by Category
* Profit by Category
* Sales by Region
* Customer Segment Analysis
* Ship Mode Analysis
* Top States by Sales
* Discount vs Profit
* Correlation Heatmap

## Key Insights

* Technology contributed significantly to total sales.
* Higher discounts generally reduced profitability.
* Sales varied across regions and customer segments.
* State-level analysis identified top-performing markets.

## Outcome

The dashboard provides a visual summary of business performance and supports data-driven decision-making through clear and informative charts.
